How to Embed

Add the widget to your website by including the following code:

<!-- Light Theme --> <wishlist-dock data-theme="light"></wishlist-dock> <!-- Dark Theme --> <wishlist-dock data-theme="dark"></wishlist-dock>

Make sure to include the widget script before the closing </body> tag:

<script type="module" src="path/to/wishlist-widget.js"></script>

Theme Toggle
